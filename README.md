# FTPClient V1.0

已完成完成。

计算机网络课程设计大作业——FTP客户端，具有图形化界面、上传、下载功能，实现断点续传功能，使用Socket方式编程，从建立TCP连接开始。使用C++语言。

实现的功能：
1.	图形化界面设计、图形化交互函数编写
2.	“连接”功能
3.	“刷新”功能
4.	“上传”功能，包含断点续传与进度条
5.	“下载”功能，包含断点续传与进度条
6.	“删除”功能

引入多线程处理，避免了后端运行时前端假死的问题
